# Cardápio Digital

Projeto simples para exibir um cardápio e gerar uma comanda compartilhável via WhatsApp.

## Como usar

1. Abra `index.html` em um navegador (pode ser arrastado ou servido por um servidor estático).
2. Selecione os itens desejados clicando no botão **+** ao lado de cada produto.
3. Clique em "Enviar pedido pelo WhatsApp" para abrir a conversa com a mensagem gerada.

## Estrutura

- `index.html` - página principal
- `style.css` - estilos básicos
- `script.js` - lógica do cardápio e geração de link WhatsApp

## Personalização

Você pode modificar o array `menu` em `script.js` para atualizar os produtos, preços e descrições.
